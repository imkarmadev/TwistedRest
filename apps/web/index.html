Built for API developers

Everything you need to design, test, and debug complex API workflows — visually.

{ }

Visual Canvas

Drag nodes, wire pins, see data flow. Two edge types: white exec edges for control flow, colored data edges for typed values.

%

Template Tokens

Type #{userId} in any URL, header, or body field. Input pins appear automatically. Wire values from upstream responses.

Z

Zod Schema Validation

Define response schemas in Zod. Output pins generate from fields. Paste sample JSON to auto-infer. One-click fix from failed responses.

||

Parallel & Sequential Loops

ForEach nodes iterate arrays with body sub-chains. Sequential for ordered processing, parallel for speed. Break Object + Convert handle the types.

!

Events & Routing

Emit/On Event for decoupled pub/sub. Match node for switch/case routing on any value — including HTTP status codes. No spaghetti wiring.

fn

Function Escape Hatch

When the visual model isn't enough, write a TypeScript transform. Typed inputs/outputs, sandboxed execution. UUID, date math, array filter — anything.

@

Environments + Auth

Per-env base URLs, headers, variables, and auth (Bearer, Basic, API Key, OAuth2 CC). Switch envs on the Start node. Dev/staging/prod in one flow.

>_

Built-in Debugging

Tap nodes show values inline. Log nodes print to the Console panel. Per-node status colors. Full request/response inspector. Stop button mid-run.

Q

Smart Palette

Right-click or press Space to search nodes. Drag a pin to empty canvas — palette filters to compatible types. Auto-wires on selection.
